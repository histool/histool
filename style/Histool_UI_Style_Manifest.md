# R&C UI Style Manifest
## 1. Visual DNA
Rooted in typographic modernism and digital brutalism, the design emphasizes **clarity, tension, and silence**.  
The interface should read like an editorial layout rather than an app.

## 2. Key References
- **Marcel Saidov** — typographic precision and philosophical restraint  
- **Hubertus Design** — archive aesthetic and narrative grid logic  
- **Chantal Jahchan** — cold romantic minimalism, asymmetrical rhythm  
- **Naranjo-Etxeberria** — black brutalist shell with poetic typographic contrast  

## 3. Core Principles
| Category | Instruction |
|-----------|-------------|
| **Color** | Black `#0A0A0A` background, white `#F5F5F5` text, cold accent `#9AE8FF`. No gradients or shadows. |
| **Typography** | Use Inter / Suisse Int’l / Neue Haas Grotesk. Optical spacing. Line height ≥ 1.7. |
| **Layout** | Based on 12-column grid. Horizontal rhythm = 80–120px. Use clear vertical zones. |
| **Interaction** | Only opacity / fade transitions (200–300ms). No easing overshoot. |
| **Texture** | Visual noise 2–5% for archival tone. |
| **Mood** | Distant yet intimate. Editorial, not commercial. Cold light, breathing silence. |

## 4. Layout Archetypes
- Landing Page → black background, centered grid, large typography block  
- Project Index → split 2-column layout: image left, metadata right  
- Project Detail → text narrative on black, white image background  
- About Page → justified paragraphs, small body text, large margins

## 5. Prohibited
- Rounded corners  
- Drop shadows  
- Gradients  
- Overlapping animations
